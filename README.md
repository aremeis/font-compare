# Font Compare

A single-page web app for comparing fonts side-by-side with pixel-level character analysis. Try it out [here](https://aremeis.github.io/font-compare).

## Features

- **20+ pre-loaded Google Fonts** plus System UI and custom font support via URL
- **Adjustable font properties**: weight, size, letter spacing, italic
- **Sample text comparison**: see both fonts rendered with the same text
- **Character comparison table**: pixel-level overlay showing differences
- **Custom fonts** persist in localStorage across sessions
